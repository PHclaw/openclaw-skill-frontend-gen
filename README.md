# openclaw-skill-frontend-gen

React / Vue 组件自动生成。根据描述或 UI 需求生成完整 TypeScript 组件代码。

## 功能

- React TS + Tailwind CSS 组件生成
- Vue 3 Composition API + Tailwind 生成
- TypeScript 严格类型（无 any）
- 可访问性（A11y）最佳实践
- 响应式设计规范

## 安装

将 `SKILL.md` 复制到 OpenClaw skills 目录即可。

## 使用场景

- "写一个组件"
- "帮我生成前端"
- 从设计稿/描述生成代码
- React / Vue 组件脚手架生成
